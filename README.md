# Cortiq Public Hub

This repository is the official public home for Cortiq documentation, releases, issues, and community discussions.

[Open Documentation](https://simeon-vanov.github.io/cortiq-public/) | [Download Latest Release](https://github.com/simeon-vanov/cortiq-public/releases/latest) | [Join Discussions](https://github.com/simeon-vanov/cortiq-public/discussions) | [Report a Bug](https://github.com/simeon-vanov/cortiq-public/issues/new/choose)

Note: This repository does not contain the application source code. It is the public-facing space for product documentation, release assets, support intake, and community feedback.

## What To Use This Repo For

- Read the official Cortiq documentation on GitHub Pages
- Report reproducible bugs in the desktop app
- Request features and product improvements
- Ask questions and share workflows in Discussions
- Download public release assets and read release notes

## What Is Not In This Repo

- Application source code
- Internal planning or engineering tasks
- Private licensing data
- Security disclosures

## Security

Do not open public issues for security problems. Report them privately to support@cortiq.trade.

## Releases

Public installers are published to GitHub Releases in this repository.

- Latest releases page: https://github.com/simeon-vanov/cortiq-public/releases/latest
- Direct Windows download: https://github.com/simeon-vanov/cortiq-public/releases/latest/download/Cortiq.msix

The public documentation site is published with GitHub Pages from the files in this repository.

- Docs site: https://simeon-vanov.github.io/cortiq-public/
